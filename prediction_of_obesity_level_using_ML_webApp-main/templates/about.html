About the Obesity Prediction Model

Introduction

The Obesity Predictor is a tool designed to help individuals assess their obesity risk based on various lifestyle and demographic factors. This tool aims to raise awareness about potential health risks associated with obesity and provides insights for healthier lifestyle choices.

Explanation of the Machine Learning Model

Our prediction model combines both traditional machine learning algorithms and deep learning techniques. The model uses:

  • Decision Trees: For quick and efficient classification based on clear decision rules.
  • Neural Networks: To capture deeper and more complex patterns in the data.

This hybrid approach ensures a higher accuracy rate by leveraging the strengths of both techniques. It identifies clear lifestyle-related patterns while accounting for complex correlations between features like age, height, weight, and family history.

Data Sources

The model was trained using publicly available datasets, ensuring a comprehensive understanding of obesity-related factors. Key data sources include:

  • Obesity Levels Dataset: Provides data on obesity trends based on various demographic and behavioral inputs.
  • Lifestyle Habits Dataset: Includes information on eating habits, physical activity, and technology use patterns.

The datasets contain anonymized information about individuals' lifestyle habits, demographics, and obesity-related factors, enabling the model to make accurate predictions.

Accuracy & Limitations

During testing, the model achieved an accuracy rate of approximately 85%. However, like any predictive model, it has its limitations:

  • Accuracy depends on the quality and completeness of user input.
  • It may not account for all factors influencing obesity, such as genetic predispositions.
  • The predictions are based on past data and may not fully reflect future trends or behaviors.

Despite these limitations, the model serves as a useful tool for preliminary assessments and awareness.
